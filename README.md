# Developer-Toolkit
A collection of utility tools for developers, consolidated from seven separate projects into a single repository. This toolkit provides a unified interface for various common programming and design tasks.

## Included Tools

This repository combines the following utilities:
<ul>
<li>Unit Converter: Convert between different units of measurement.</li>
<li>Text Case Converter: Change text formatting (uppercase, lowercase, title case, etc.).</li>
<li>Base64 Encoder / Decoder: Encode text to Base64 and decode it back.</li>
<li>URL Encoder / Decoder: Standard URL encoding and decoding.</li>
<li>JSON/XML Viewer & Formatter: Parse, view, and beautify JSON and XML data.</li>
<li>Color Palette Generator: Generate and visualize color schemes.</li>
<li>Image Resizer & Compressor: Resize and compress images directly in the browser.</li>
</ul>
